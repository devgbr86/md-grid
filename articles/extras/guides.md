# Guias e Referências para Desenvolvedores

Este documento reúne links de referência, documentação oficial, ferramentas e guias de aprendizado para desenvolvedores web, com foco especial em desenvolvimento frontend. O objetivo é centralizar em um único lugar os recursos mais úteis para consulta no dia a dia e para aprofundamento técnico.

---

## Documentacao oficial

As fontes primárias são sempre o ponto de partida mais confiável. Antes de buscar tutoriais de terceiros, vale consultar a documentação oficial da tecnologia em questão.

| Tecnologia | Link |
|---|---|
| HTML — referência completa de elementos e atributos | https://developer.mozilla.org/pt-BR/docs/Web/HTML |
| CSS — referência de propriedades e valores | https://developer.mozilla.org/pt-BR/docs/Web/CSS |
| JavaScript — linguagem, APIs e ECMAScript | https://developer.mozilla.org/pt-BR/docs/Web/JavaScript |
| Web APIs — Canvas, Fetch, Storage, WebGL e mais | https://developer.mozilla.org/pt-BR/docs/Web/API |
| TypeScript — documentação oficial | https://www.typescriptlang.org/docs/ |
| Node.js — documentação oficial | https://nodejs.org/en/docs |
| Git — referência de comandos | https://git-scm.com/doc |
